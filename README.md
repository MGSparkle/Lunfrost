# Lunfrost
霜月脑机协同概率模拟系统，可根据电极推导手势，或根据手势推导电极
Lunfrost 脑机协同仿真平台 EXE 功能简述
一、电极可视化
绘制标准 10-20 头皮全电极拓扑图，支持窗口自适应缩放
鼠标点击 / 输入框批量选中电极，带脉冲、连线动态动画
一键重置清空所有电极选择
二、正向推理（电极→动作）
选取任意电极组合，自动计算 6 类抓取动作识别概率
进度条可视化，最高概率动作绿色高亮区分
三、逆向推理（动作→电极）
勾选目标手部动作，反向筛选敏感度 TOP8 电极
电极按贡献度分橙 / 蓝两级上色，拓扑图同步高亮
四、底层兼容容错
内置动作 - 电极概率数据表，打包 EXE 自带资源
丢失数据文件时自动生成随机基准数据，程序不崩溃
兼容单文件打包，图标、图表资源自动识别路径
